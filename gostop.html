🎴 화투 규칙

목표

7점 이상 얻으면 스톱을 선언할 수 있습니다. 고를 외치면 계속 진행해 더 높은 점수를 노릴 수 있습니다.

패 종류 & 점수

  • 광(光) — 1, 3, 8, 11, 12월. 3광=3점, 4광=4점, 5광=15점
  • 띠(短冊) — 청단(3,6,9월)/홍단(1,2,3월)/초단(4,5,7월)=각 3점. 띠 5장이상=1점씩 추가
  • 동물(動物) — 고도리(2,4,8월 새)=5점. 동물 5장이상=1점씩 추가
  • 피(皮) — 10장=1점, 이후 1장당 1점 추가

진행

  • 손패에서 바닥 패와 같은 월의 카드를 냅니다.
  • 같은 월이 있으면 가져옵니다. 없으면 버립니다.
  • 덱에서 1장 뒤집어 같은 월이면 가져옵니다.
  • 같은 월이 3장이면 모두 가져옵니다(폭탄).

고/스톱

  • 7점 이상: 스톱 선언 가능 (베팅 칩 획득)
  • 고: 계속 진행, 점수 2배 보너스 가능
  • 3고 이상: 상대방이 스톱해도 배당 증가

목표

화투 2장으로 끝수 합을 겨루는 베팅 게임입니다.

패 순위 (높은 순)

  • 38광땡 — 3월광 + 8월광 (최고패)
  • 18광땡 — 1월광 + 8월광
  • 13광땡 — 1월광 + 3월광
  • 9땡 ~ 1땡 — 같은 월 2장
  • 알리 — 1월 + 2월
  • 독사 — 1월 + 4월
  • 구삥 — 1월 + 9월
  • 장삥 — 1월 + 10월
  • 장사 — 4월 + 10월
  • 세륙 — 4월 + 6월
  • 끗 — 두 장 끝자리 합 (9끗 최고)
  • 망통 — 합이 0 (최저)

진행

  • 2장씩 받고 베팅: 삥(기본) / 따당(2배) / 콜 / 다이(포기)
  • 모두 콜하면 패 공개 → 높은 패 승리
